SB Banner Widget is a plugin that allows to add banner widget on your WordPress site.

== Description ==

SB Banner Widget is a plugin that allows to add banner widget on your WordPress site. Drag SB Banner into widget area where you want banner will be shown.

**Required Plugin**

* [SB Core](https://wordpress.org/plugins/sb-core/)

**Translations**

* English
* Vietnamese

== Installation ==

Install this plugin from your WordPress site Dashboard or follow these steps below:

1. Download plugin from WordPress Plugins directory and extract it.
1. Upload the `sb-banner-widget` folder to the `/wp-content/plugins/` directory.
1. Activate the SB Banner Widget plugin through the 'Plugins' menu in WordPress.
